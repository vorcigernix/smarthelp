# Smartsupp — OpenCart
## Non stiamo più sviluppando il plugin per OpenCart, quindi per alcune versioni questo plugin potrebbe non funzionare affatto o potrebbe non funzionare corre
Non stiamo più sviluppando il plugin per OpenCart, quindi per alcune versioni questo plugin potrebbe non funzionare affatto o potrebbe non funzionare correttamente. In questi casi sarà necessario inserire manualmente il codice della chat per utilizzare Smartsupp.
Integrazione della Live Chat di Smartsupp
OpenCart 3.0
Scegli tutorial: OpenCart 3.0, OpenCart 2.0, Opencart 1.5
Scarica listensione
Scarica l'estensione 3.0 in file .ZIP.
Carica l'estensione
Extensions > Installer > Upload
Carica il .ZIP, viene installato automaticamente.
Installa l'estensione
Extensions > Extensions > scegli Modules > trova Smartsupp 
Adesso clicca su Edit. Se non è possibile clicca su Install, aggiorna la pagina e clicca su Edit di nuovo.
Accedi all'account Smartsupp
Puoi Creare un account gratis oppure Conettere un account già esistente.
Se tutto è a posto vedi questa pagina:
Aggiorna la cache
Extensions > Modifications e clicca sull'icona blu Refresh.
Apri la homepage del tuo sito web, aggiorna il browser e la chat box sarà visibile.
Problemi conosciuti
Ci sono alcuni problemi noti in specifiche versioni di OpenCart che possono influire sul corretto funzionamento di alcune funzioni. Di seguito forniamo soluzioni per queste versioni.
3.0.3.5, 3.0.3.6
Esiste un bug in queste versioni che impedisce ai plugin installati di modificare il codice del sito web (nel nostro caso: inserire il codice Smartsupp sul sito web). Le possibili soluzioni sono:
aggiorna OpenCart alla versione 3.0.3.7 o successiva: il bug è stato risolto lì
usa una versione precedente di OpenCart (3.0.3.3 o inferiore) - il bug non esiste qui
persone tecniche come gli sviluppatori possono modificare il file di modifica di OpenCart in base a questo problema

Adesso clicca su Edit. Se non è possibile clicca su Install, aggiorna la pagina e clicca su Edit di nuovo.
Accedi all'account Smartsupp
Puoi Creare un account gratis oppure Connettere un account già esistente.
Se tutto è a posto vedi questa pagina:
Aggiorna la cache
Extensions > Modifications a clicca sull'icona blu Refresh.
Apri la homepage del tuo sito, aggiorna il browser e la chat box sarà visibile.
OpenCart 1.5
Scegli tutorial: OpenCart 3.0, OpenCart 2.0, Opencart 1.5
Scarica l'estensione
Scarica l'estensione 1.5 in file .ZIP.
Installa l'estensione
Apri il file. Carica cartelle admin + catalog al tuo server usando cliente FTP client.
Extensions > Modules, trova Smartsupp Live Chat e clicca su Install. 
Aggiorna la pagina web e adesso clicca su Edit.
Inserisci l'ID della chat di Smartsupp
Il tuo codice è in Smartsupp > Settings > Chat box > Chat code.
Inserisci l'ID della chat di Smartsupp, cioè il codice a riga singola che vedi nella dashboard di Smartsupp.
Clicca su Add Module e utilizza le stesse impostazioni che vedi sotto il #3 nello screenshot e clicca su Save.
Sei ancora confuso? Contattaci tramite la chat box nella tua dashboard e saremo felici di aiutarti. Si prega di notare che il servizio clienti è incluso solo nei pacchetti a pagamento.
